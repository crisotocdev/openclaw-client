1️⃣ Abrir las apps necesarias

Abrir estas 3 cosas:

1️⃣ VSCode

Abrir el proyecto:

reactNativeMoltbot

y también si lo tienes separado:

moltbot_server
2️⃣ Terminal backend (Rust)

Ir a la carpeta del servidor:

moltbot_server

y ejecutar:

cargo run

Debe mostrar algo como:

Server listening on 0.0.0.0:8080
3️⃣ Terminal frontend (React Native)

Ir a:

reactNativeMoltbot

y ejecutar:

npm start

o

npx expo start

Luego abrir:

http://localhost:8081
2️⃣ Verificar que todo esté funcionando

En el panel probar:

PING
TIME
WHOAMI
PROCESOS

Confirmar que:

backend aparece ONLINE

historial inteligente funciona

botón Repetir funciona

3️⃣ Implementar comando STATUS (Backend Rust)

Archivo a modificar:

moltbot_server/src/commands.rs

Agregar comando:

STATUS

Respuesta esperada:

SERVER ONLINE
VERSION 0.1.0
UPTIME <tiempo>
4️⃣ Implementar comando HELP (Backend Rust)
